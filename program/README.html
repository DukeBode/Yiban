  1. 导出上月1日至本月1日的轻应用数据
  2. 筛选数据选定合适的时间
  3. pv改为浏览量，按浏览量排名选取前五，校级不得超过两条

  5. 整理内容，以序号、机构、轻应用名称、添加人数、浏览量的顺序整理
  6. 机构根据用户姓名及应用名称填写，以 LUT 的院系设置为准

  5. $分列，筛选，排序
